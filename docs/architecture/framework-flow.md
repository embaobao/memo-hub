                              │

        │                     │                     │

## 🔄 记忆写入流程

```
用户执行: memohub add "知识内容" -c category -t tags
      │
      ▼
┌─────────────────────────────┐
│ 1. CLI 解析参数             │
│    - 内容文本                │
│    - 分类 (category)         │
│    - 标签 (tags)             │
│    - 重要性 (importance)     │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 2. 构建 Text2MemInstruction │
│    {                        │
│      op: MemoOp.ADD,        │
│      track: 'track-insight',│
│      data: { content, ... } │
│    }                        │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 3. MemoryKernel 分发        │
│    kernel.execute(instruction)│
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 4. InsightTrack 处理        │
│    track.execute(instruction)│
│    - 解构指令                │
│    - 准备数据                │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 5. 内容处理                 │
│    - 生成 ID (UUID)          │
│    - 提取元数据              │
│    - 计算哈希                │
└─────────────────────────────┘
      │
      ├──────────────────┐
      ▼                  ▼
┌──────────────┐  ┌──────────────┐
│ CAS 存储     │  │ AI 嵌入      │
│ 存储原始内容  │  │ 生成向量     │
└──────────────┘  └──────────────┘
      │                  │
      │                  ▼
      │          ┌──────────────┐
      │          │ 768 维向量   │
      │          └──────────────┘
      │                  │
      └──────────────────┘
              │
              ▼
┌─────────────────────────────┐
│ 6. 向量存储 (LanceDB)       │
│    - 存储向量                │
│    - 存储元数据              │
│    - 建立索引                │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 7. 返回结果                 │
│    {                        │
│      success: true,         │
│      id: "uuid",            │
│      hash: "sha256"         │
│    }                        │
└─────────────────────────────┘
```

## 🔍 记忆检索流程

```
用户执行: memohub search "查询内容" -l 5
      │
      ▼
┌─────────────────────────────┐
│ 1. CLI 解析查询             │
│    - 查询文本                │
│    - 返回数量 (limit)        │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 2. 构建 Text2MemInstruction │
│    {                        │
│      op: MemoOp.SEARCH,     │
│      track: 'track-insight',│
│      data: { query, limit } │
│    }                        │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 3. MemoryKernel 分发        │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 4. InsightTrack 处理        │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 5. AI 嵌入查询              │
│    将查询文本转为 768 维向量│
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 6. 向量检索 (LanceDB)       │
│    - 计算相似度              │
│    - 返回 Top K 结果         │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 7. CAS 存储取回内容         │
│    根据 hash 读取原始内容   │
└─────────────────────────────┐
      │
      ▼
┌─────────────────────────────┐
│ 8. 返回结果                 │
│    {                        │
│      success: true,         │
│      results: [{            │
│        content,             │
│        score,               │
│        metadata             │
│      }]                     │
│    }                        │
└─────────────────────────────┘
```

## 🚀 统一检索流程 (search-all)

```
用户执行: memohub search-all "查询"
      │
      ▼
┌─────────────────────────────┐
│ 1. MemoryKernel 接收        │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 2. Librarian 触发检索流水线 │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 3. 双轨并行检索             │
│    ├─ track-insight 检索    │
│    └─ track-source 检索     │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 4. 结果融合                 │
│    - 合并相似结果            │
│    - 重新排序                │
│    - 去重过滤                │
└─────────────────────────────┐
      │
      ▼
┌─────────────────────────────┐
│ 5. 返回统一结果             │
└─────────────────────────────┘
```

## 🧹 治理流程

```
去重流程 (memohub dedup)
      │
      ▼
┌─────────────────────────────┐
│ 1. Librarian 触发去重       │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 2. 读取轨道数据             │
│    - 批量读取向量            │
│    - 计算相似度矩阵          │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 3. 检测重复项               │
│    - 相似度 > 阈值           │
│    - 标记重复组              │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 4. 生成去重报告             │
│    - 重复项列表              │
│    - 建议删除项              │
└─────────────────────────────┘


蒸馏流程 (memohub distill)
      │
      ▼
┌─────────────────────────────┐
│ 1. Librarian 触发蒸馏       │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 2. 读取轨道数据             │
│    - 获取所有知识条目        │
│    - 按分类分组              │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 3. 调用 AI 总结             │
│    - 按分类生成摘要          │
│    - 提取关键概念            │
└─────────────────────────────┘
      │
      ▼
┌─────────────────────────────┐
│ 4. 保存蒸馏结果             │
│    - 存储摘要                │
│    - 建立索引                │
└─────────────────────────────┘
```
