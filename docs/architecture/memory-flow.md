# MemoHub v3 记忆流程图

## 🎯 记忆生命周期

```
┌─────────────────────────────────────────────────────────────────┐
│                        记忆生命周期                              │
└─────────────────────────────────────────────────────────────────┘

  [输入]       [处理]        [存储]        [检索]        [治理]
    │            │            │            │            │
    ▼            ▼            ▼            ▼            ▼
┌─────┐      ┌─────┐      ┌─────┐      ┌─────┐      ┌─────┐
│文本/│      │嵌入/│      │CAS/ │      │语义/│      │去重/│
│代码 │  →   │元数据│ →   │向量 │  →   │召回│  →   │蒸馏│
└─────┘      └─────┘      └─────┘      └─────┘      └─────┘
```

## 📝 知识记忆流程 (track-insight)

### 1. 添加知识

```
用户输入: memohub add "知识内容" -c category -t tag1,tag2
      │
      ▼
┌──────────────────────────────────────────────────────┐
│ Step 1: CLI 参数解析                                 │
│ ┌────────────────────────────────────────────────┐  │
```

## 🧹 治理流程

### 1. 去重流程 (dedup)

```
用户命令: memohub dedup --track track-insight
      │
      ▼
┌──────────────────────────────────────────────────────┐
│ Step 1: Librarian 启动去重                            │
│ op: MemoOp.DEDUP                                      │
└──────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────┐
│ Step 2: 读取轨道数据                                  │
│ ┌────────────────────────────────────────────────┐  │
│ │ 1. 获取所有向量: [v1, v2, v3, ..., vN]         │  │
│ │ 2. 批量读取向量 (batch_size=100)               │  │
│ └────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────┐
│ Step 3: 计算相似度矩阵                                │
│ ┌────────────────────────────────────────────────┐  │
│ │ for i in 0..N:                                 │  │
│ │   for j in i+1..N:                             │  │
│ │     similarity[i,j] = cosine(vi, vj)           │  │
│ │                                                 │  │
│ │ 示例:                                           │  │
│ │ ┌─────────┬─────────┬─────────┐                │  │
│ │ │         │  v2     │  v3     │                │  │
│ │ ├─────────┼─────────┼─────────┤                │  │
│ │ │ v1      │ 0.95    │ 0.23    │                │  │
│ │ ├─────────┼─────────┼─────────┤                │  │
│ │ │ v2      │ -       │ 0.31    │                │  │
│ │ └─────────┴─────────┴─────────┘                │  │
│ └────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────┐
│ Step 4: 检测重复组                                    │
│ ┌────────────────────────────────────────────────┐  │
│ │ 阈值: 0.90                                      │  │
│ │                                                 │  │
│ │ 重复组:                                         │  │
│ │ - Group 1: [v1, v2] (similarity: 0.95)        │  │
│ │ - Group 2: [...]                               │  │
│ └────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────┘
      │
      ▼
┌──────────────────────────────────────────────────────┐
│ Step 5: 生成去重报告                                  │
│ ┌────────────────────────────────────────────────┐  │
│ │ {                                              │  │
│ │   total: 100,                                  │  │
│ │   duplicates: 5,                               │  │
│ │   groups: [                                    │  │
│ │     {                                          │  │
│ │       id: "group-1",                           │  │
│ │       items: ["id1", "id2"],                   │  │
│ │       similarity: 0.95,                        │  │
│ │       suggestion: "delete id2"                 │  │
│ │     },                                         │  │
│ │     ...                                        │  │
│ │   ]                                            │  │
│ │ }                                              │  │
│ └────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────┘
```

## 📊 数据结构映射

### CAS 存储结构

```
~/.memohub/blobs/
├── aa/
│   ├── aab3c4d5e6f7... (原始内容)
│   └── aad8e9f0a1b2... (原始内容)
├── bb/
│   └── bbc3d4e5f6a7... (原始内容)
└── ...

每个文件名 = SHA256 哈希
文件内容 = 原始文本/代码
```

### LanceDB 表结构

#### insight 表

```typescript
{
  id: string,                    // UUID
  vector: number[768],          // 嵌入向量
  metadata: {
    hash: string,               // CAS 哈希
```
