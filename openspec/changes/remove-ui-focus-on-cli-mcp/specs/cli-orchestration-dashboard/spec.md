## ADDED Requirements

### Requirement: TUI Interface
The system SHALL provide a terminal-based dashboard (`memohub dashboard`) for real-time monitoring.

### Requirement: Flow Visualizer
The dashboard SHALL display the current execution tree of a dispatched instruction using ASCII or box-drawing characters.

### Requirement: Hot-Reload Toggle
Users SHALL be able to reload the `config.jsonc` from the TUI without restarting the daemon.
