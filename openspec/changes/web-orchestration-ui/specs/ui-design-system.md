# Ether UI 设计系统规格 (Design System Spec)

## 🎨 视觉理念: "Hardcore Minimalism & Natural Materials"
Ether UI 的核心在于平衡“硬核工程”与“自然材质”。它支持深浅双色模式，并具备 iOS 级的动态玻璃拟态效果。

## 1. 色彩与模式 (The Dual-Theme)

| 元素 | 深色模式 (Dark) | 浅色模式 (Light) |
| :--- | :--- | :--- |
| **画布底色** | `#000000` | `#FFFFFF` |
| **节点容器** | `#09090B (Zinc-950)` | `#F4F4F5 (Zinc-100)` |
| **边框颜色** | `rgba(255, 255, 255, 0.08)` | `rgba(0, 0, 0, 0.06)` |
| **主要文字** | `#FAFAFA` | `#09090B` |
| **次要文字** | `#A1A1AA` | `#71717A` |

## 2. iOS 级玻璃拟态 (Advanced Material Design)

### 2.1 亚克力材质栈 (The Acrylic Stack)
所有浮窗（Overlay）和侧边栏（Sidebar）必须遵循以下分层渲染逻辑：
1. **背景采样**: 实时读取下方内容的像素。
2. **多重模糊**: `backdrop-filter: blur(50px) saturate(180%) contrast(100%)`。
3. **反射高光**: 顶层增加 `linear-gradient(135deg, rgba(255,255,255,0.05), transparent)`。
4. **噪声遮罩**: 叠叠加一层 2% 不透明度的灰度颗粒，模拟胶片质感。

### 2.2 动态模糊转换 (Motion Blur)
- **物理动画**: 页面切换和侧边栏滑出时，背景模糊度随位移非线性变化（Lerp）。
- **边缘衍射**: 窗口四周具备极细的 `ring` 高光，使其在深浅色背景下都能清晰分离。

## 3. 核心组件交互

### 3.1 节点脉冲 (Flow Pulse)
- **视觉**: 节点外边框产生向外扩散的波纹，中心点产生微弱的颜色呼吸效果。
- **iOS 优化**: 扩散波纹具备“果冻”般的弹性，使用 `framer-motion` 的 `type: "spring"` 驱动。

### 3.2 跨 Workspace 空间感
- **叠纸切换**: 切换 Workspace 时，当前空间沿 Z 轴向后退避并增加模糊，新空间由下方平滑升起。

## 4. 验收准则
- 所有的交互延迟必须感知为 0 (通过影子配置实现)。
- 界面在 200% 缩放（Retina）下必须保持像素对齐，无毛刺。
- 动画曲线统一使用 `[0.23, 1, 0.32, 1]`。
